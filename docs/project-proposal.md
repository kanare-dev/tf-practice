# プロジェクト提案書

## 推奨プロジェクト: パーソナルノート/メモアプリ

### なぜこのプロジェクトが適切か

- **難易度**: 簡単すぎず、複雑すぎない
- **学習要素**: 使用予定のAWSサービスを適切に活用できる
- **拡張性**: MVPから段階的に機能追加可能
- **実用性**: 実際に使えるアプリケーション

### プロジェクト概要

個人用のノート/メモ管理アプリケーション。マークダウン形式のメモを作成・編集・保存・検索できる。

## MVP（最小機能）

### 機能要件

1. **ユーザー認証**
   - Cognito でのサインアップ/サインイン
   - セッション管理

2. **ノートのCRUD操作**
   - ノートの作成（タイトル、本文、作成日時）
   - ノート一覧の取得
   - ノートの詳細取得
   - ノートの更新
   - ノートの削除

3. **基本的なUI**
   - シンプルなフロントエンド（HTML/CSS/JavaScript）
   - S3 で静的ホスティング

### データモデル（DynamoDB）

```
NoteTable
- userId (Hash Key)     : ユーザーID（Cognitoのsub）
- noteId (Range Key)    : ノートID（UUID）
- title                 : タイトル
- content               : 本文（マークダウン）
- createdAt             : 作成日時
- updatedAt             : 更新日時
- tags                  : タグ（配列、オプション）
```

### API エンドポイント

- `GET /notes` - ユーザーのノート一覧取得
- `POST /notes` - ノート作成
- `GET /notes/{noteId}` - ノート詳細取得
- `PUT /notes/{noteId}` - ノート更新
- `DELETE /notes/{noteId}` - ノート削除

## Phase 2: 拡張機能（将来的に追加）

1. **検索機能**
   - DynamoDB GSI を使用した全文検索
   - タグによるフィルタリング

2. **ファイル添付**
   - S3 への画像/ファイルアップロード
   - 署名付きURL での安全なダウンロード

3. **共有機能**
   - ノートの共有リンク生成
   - 公開/非公開設定

4. **カテゴリ/フォルダ機能**
   - ノートの分類
   - 階層構造のサポート

## 学習できるAWSサービスと機能

### MVP段階

- **Cognito**: ユーザー認証・認可、User Pool
- **API Gateway**: RESTful API、Cognito Authorizer統合
- **Lambda**: サーバーレスバックエンド、環境変数
- **DynamoDB**: NoSQLデータベース、クエリ、GSI（将来的に）
- **S3**: 静的ファイルホスティング、バケットポリシー

### 拡張段階

- **S3**: ファイルストレージ、署名付きURL
- **DynamoDB GSI**: 複合クエリ、ソート
- **Step Functions**: ワークフローオーケストレーション（バッチ処理など）
- **CodePipeline/CodeBuild**: CI/CDパイプライン
- **CloudWatch**: ログとメトリクス

## 実装の複雑度

### 低（MVP）

- 基本的なCRUD操作
- シンプルな認証フロー
- 静的UI

### 中（Phase 2）

- 検索機能の実装
- ファイルアップロード処理
- 複雑なクエリパターン

### 高（Phase 3以降）

- リアルタイム同期
- 高度な権限管理
- パフォーマンス最適化

## 代替案

### 案1: シンプルなタスク管理アプリ

- **特徴**: Todoアプリの拡張版
- **難易度**: ⭐⭐（少し簡単）
- **学習要素**: プロジェクト、優先度、期限管理

### 案2: ファイル共有アプリ（Dropbox風）

- **特徴**: ファイルのアップロード・ダウンロード・共有
- **難易度**: ⭐⭐⭐（適切）
- **学習要素**: S3の詳細機能、署名付きURL、フォルダ構造

### 案3: シンプルなブログ/記事管理アプリ

- **特徴**: 記事の作成・公開・管理
- **難易度**: ⭐⭐⭐（適切）
- **学習要素**: 公開/非公開管理、カテゴリ、コメント機能

## 推奨: パーソナルノート/メモアプリを選ぶ理由

1. **バランスが良い**: 基本的な機能から始められ、段階的に拡張可能
2. **学習効果が高い**: 主要なAWSサービスを実践的に学習できる
3. **実用性がある**: 実際に使えるアプリケーションを作れる
4. **拡張性がある**: 将来、他のサービス（Step Functions、EventBridgeなど）を追加しやすい
